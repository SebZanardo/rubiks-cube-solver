__FEATURES:__
* Validates any 3x3 cube position. Uses edge and corner parity tests as well as permutation parity test

__CONTROLS:__  
* `1-6` Change active paint colour
* `LEFT_CLICK` Paint active colour on hovered cube tile
* `F` Rotate Front
* `R` Rotate Right
* `U` Rotate Up
* `B` Rotate Back
* `L` Rotate Left
* `D` Rotate Down
* `L_SHIFT (HOLD)` Rotate Prime (rotate face anticlockwise)
* `L_CONTROL (HOLD)` Rotate Double (rotate face 180 degrees)
