__CONTROLS:__  
* `1-6` Change active paint colour
* `LEFT_CLICK` Paint active colour
* `F` Rotate Front
* `R` Rotate Right
* `U` Rotate Up
* `B` Rotate Back
* `L` Rotate Left
* `D` Rotate Down
* `L_SHIFT (HOLD)` Rotate Prime (Anticlockwise)
* `L_CONTROL (HOLD)` Rotate Double (rotate)
